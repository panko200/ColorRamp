カラーランプ
製作者：Panko200
配布場所：https://github.com/panko200/ColorRamp

## 概要

黒から白にかけてのグラデーションをもとに、別のパターンの色に変更ができるエフェクトです。

## 使用方法

映像エフェクト「カラーランプ」が追加されます。  
グラデーションになっている所をクリックすると、カラーストップが追加されます。カラーストップを削除する場合はグラデーションの下の色が入っているところの右にあるゴミ箱ボタン 🗑 を押してください。また、カラーストップは左ドラッグで自由に移動できます。

## アンインストール方法

1. YMM4 を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4 を終了する。
3. `ColorRampEffect`という名前のフォルダを削除する。

## 注意点
ゆっくりムービーメーカーv4.47.0.0以上を使用してください。
